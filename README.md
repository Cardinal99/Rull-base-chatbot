# 🤖 Rull-Base Chatbot

A **rule-based natural language processing chatbot** built in Python that demonstrates intent detection through keyword matching and tokenization.

## 📋 Description

Rull-Base Chatbot is a terminal-based conversational agent that processes user input through text tokenization and pattern matching. It responds to predefined intents using a dictionary-based lookup system with randomized reply variations. This project showcases fundamental NLP concepts including text cleaning, tokenization, and rule-based response generation.

## ✨ Features

- **Text Tokenization**: Converts input to lowercase and removes punctuation
- **Intent Detection**: Matches keywords against a predefined intent dictionary
- **Randomized Responses**: Multiple reply variations for each intent
- **Colorful Terminal UI**: Enhanced visual experience with ANSI color codes
- **Cross-Platform Support**: Works on Windows, macOS, and Linux
- **Full-Screen Menu**: Clean, centered interface with colored backgrounds
